# 全局逻辑自查表

该文档记录了 `README.md` 中八大维度的状态判定、资产建议逻辑及其合理性分析。它是测试脚本的**唯一真理来源**。

## 1. 图标图例
*   ⛔ **否决 (Veto)**: 绝对禁止/建议卖出。针对所有不适合当前环境的资产（如`风险类`、`防御类`）
*   ⚠️ **警惕 (Warn)**: 可以持有但需观察/停止加仓。针对有潜在风险的资产（如特定`风险类`、`防御类`）
*   ⚔️ **进攻 (Attack)**: 推荐买入/持有的资产。主要针对`风险类`
*   🛡️ **避险 (Defend)**: 推荐买入/持有的资产。主要针对`防御类`、`避险类`和`现金类`。

## 2. 状态互斥矩阵 (针对同一资产)

| 状态 A | 状态 B | 共存可行性 | 逻辑解释 |
| :--- | :--- | :--- | :--- |
| **⛔ 否决** | 任何其他 | **❌ 互斥** | 否决具有最高优先级。一旦否决，其他建议无效。 |
| **⚠️ 警惕** | **⚔️ 进攻** | **❌ 互斥** | 如果要警惕，就不能作为主力进攻方向。应降级为“观察”或移出进攻列表。 |
| **⚠️ 警惕** | **🛡️ 避险** | **✅ 共存** | 可以推荐避险，同时提示该避险资产的潜在风险（如美债波动）。 |
| **⚔️ 进攻** | **🛡️ 避险** | **❌ 互斥** | 角色定位冲突。应根据当前宏观环境界定它是“矛”还是“盾”。(黄金除外，视周期而定) |

## 3. 最终建议生成机制 (四池分流)

最终建议由以下四个逻辑池依次筛选生成。系统必须确保资产在池子间的流动符合以下规则：

### 3.1 基础池定义
*   **否决池**: 包含所有被 `⛔[否决]` 的资产。
    *   *效力*: **最高优先级**。进入此池的资产，永久丧失进入另外所有池子的资格。
*   **警惕池**: 包含所有被 `⚠️[警惕]` 的资产。
    *   *效力*: **次高优先级**。进入此池的资产，不能进入进攻池，但可以保留在避险池（带病避险）。

### 3.2 最终建议池生成
*   **⚔️ 进攻池**
    *   *定义*: 真正值得买入的获利资产。
    *   *规则*: 必须至少被一个维度推荐 `⚔️`，且 **绝不能** 出现在否决池或警惕池中。
    *   *公式*: `进攻池 = (所有推荐 ⚔️) - 否决池 - 警惕池`
*   **🛡️ 避险池**
    *   *定义*: 用于对冲风险或资金停泊的资产。
    *   *规则*: 必须至少被一个维度推荐 `🛡️`，且 **绝不能** 出现在否决池中。
    *   *特权*: **允许** 包含警惕池中的资产（即：虽然有瑕疵，但为了避险可以接受）。
    *   *公式*: `避险池 = (所有推荐 🛡️) - 否决池`

### 3.3 冲突裁决 (Conflict Resolution)
*   如果一个资产同时满足“进攻”和“避险”条件（如黄金）：
    *   **原则**: 视宏观环境而定。通常 **避险优先级 > 进攻优先级**（保命第一）。
    *   *处理*: 优先归入避险池，除非该资产被明确标记为“进攻主力”（如负利率时的黄金）。

## 6. 附录：核心术语字典

*   **久期敏感资产**: 对利率变化非常敏感的资产。利率涨它就跌，利率跌它就涨。代表：**长债 (`TLT`)**、**生物医药 (`XBI`)**、**未盈利科技股**。
*   **高弹性资产**: 波动率大，涨跌都比大盘凶猛的资产。代表：**加密货币 (`BTC`)**、**纳指三倍做多 (`TQQQ`)**、**小盘股 (`IWM`)**。
*   **强周期板块**: 业绩随经济周期剧烈波动的行业。经济好时赚大钱，差时亏大钱。代表：**能源 (`XLE`)**、**原材料 (`XLB`)**、**工业 (`XLI`)**、**金融 (`XLF`)**。
*   **高估值资产**: 价格透支了未来预期的资产，通常指科技股。代表：**科技股 (`QQQ`)**、**加密货币**。
*   **重资产**: 需要投入大量资金购买设备、厂房的行业。代表：**房地产**、**基建**、**钢铁**。
*   **中下游制造**: 生产终端消费品或中间品的制造业。代表：**家电**、**汽车**、**机械**。
*   **上游资源**: 开采原材料的行业。代表：**煤炭**、**石油**、**铜矿**、**锂矿**。
*   **投机股**: 缺乏业绩支撑，靠题材炒作的股票。代表：**Meme股**、**垃圾股**。
*   **绩优股/核心资产 (Quality/Core)**: 业绩稳定增长，行业地位稳固的龙头股。代表：**标普500核心成分股 (`AAPL`, `MSFT`)**、**红利贵族**。
*   **反转股**: 之前跌得很惨，但基本面开始好转的股票。代表：**困境反转的行业龙头**。
*   **低效资产**: 在特定环境下回报率极低甚至亏损的资产。例如：通胀时的**现金**（购买力贬值）、牛市时的**债券**（跑输大盘）。
*   **出海创汇**: 收入主要来自海外，赚美元利润的中国企业。受益于海外需求复苏。代表：**家电出口、工程机械、跨境电商**。
*   **行业龙头**: 市场占有率第一，拥有规模优势，能熬死竞争对手的企业。代表：**各细分赛道的"茅"**。
*   **提价权**: 即使涨价，客户也不敢跑的护城河。通常指**高端白酒、奢侈品、垄断资源**。
*   **美债 vs 国债**: 本看板中，在涉及美国宏观指标（如EFFR/10Y-2Y/OAS）时，统一称为**美债**。只有在涉及中国宏观指标时，才可能指中国国债。
*   **大宗**: **工业大宗商品 (铜/油)**，不包含避险属性的黄金。
