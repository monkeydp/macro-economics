# 全局逻辑自查表

该文档记录了 `README.md` 中八大维度的状态判定、资产建议逻辑及其合理性分析。

## 图标图例
*   ⛔ **否决 (Veto)**: 绝对禁止/建议卖出。针对所有不适合当前环境的资产（如风险类、防御类）
*   ⚠️ **警惕 (Warn)**: 可以持有但需观察/停止加仓。针对有潜在风险的资产（如特定风险类、防御类）
*   ⚔️ **进攻 (Attack)**: 推荐买入/持有的资产。主要针对风险类
*   🛡️ **避险 (Defend)**: 推荐买入/持有的资产。主要针对防御类、避险类和现金类。

## 状态互斥矩阵 (针对同一资产)

| 状态 A | 状态 B | 共存可行性 | 逻辑解释 |
| :--- | :--- | :--- | :--- |
| **⛔ 否决** | 任何其他 | **❌ 互斥** | 否决具有最高优先级。一旦否决，其他建议无效。 |
| **⚠️ 警惕** | **⚔️ 进攻** | **❌ 互斥** | 如果要警惕，就不能作为主力进攻方向。应降级为“观察”或移出进攻列表。 |
| **⚠️ 警惕** | **🛡️ 避险** | **✅ 共存** | 可以推荐避险，同时提示该避险资产的潜在风险（如美债波动）。 |
| **⚔️ 进攻** | **🛡️ 避险** | **❌ 互斥** | 角色定位冲突。应根据当前宏观环境界定它是“矛”还是“盾”。(黄金除外，视周期而定) |
