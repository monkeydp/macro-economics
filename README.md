# [薯仔] 宏观经济审计看板 - 投资者小白生存指南

## 1. 写在前面：为什么你需要这个看板？

投资世界里充满了噪音。新闻说“经济很好”，股市却大跌；专家说“要降息”，手里的基金却还在亏。

这个看板的设计初衷，就是**剥离所有噪音，只看最核心的“水流”和“天气”**。我们将复杂的宏观经济简化为一个**“生态鱼塘”模型**，帮助你判断：
- **现在适合养鱼吗？**（进攻/买入）
- **还是暴风雨要来了？**（防守/卖出）

---

## 2. 核心概念：宏观经济的“生态鱼塘”比喻

想象你是一个鱼塘主（投资者），你的钱就是鱼。鱼能不能养好，取决于以下 8 个维度的环境指标：

| 维度 | 指标名称 | 鱼塘比喻 | 解释 |
| :--- | :--- | :--- | :--- |
| **1** | **基准利率 (EFFR)** | **中央水龙头** | 央行控制水流的总开关。利率越低，水流越大（钱越便宜），鱼塘水位越高。 |
| **2** | **真实利率 (TIPS)** | **地心引力** | 决定资产起飞的难度。引力越大（利率高），资产价格越难涨；引力小，猪都能上天。 |
| **3** | **利差结构 (10Y-2Y)** | **输水管道** | 管道是否通畅。正常水往低处流（长债利率>短债）。如果倒流（倒挂）或爆管（牛陡），说明系统出故障了。 |
| **4** | **信用风险 (OAS)** | **河道淤泥** | 银行愿不愿意借钱给企业。淤泥多（利差大），水流不过去，企业会渴死。 |
| **5** | **流动性 (M2)** | **蓄水池水位** | 整个市场的钱是多了还是少了。水位上涨，鱼才欢腾。 |
| **6** | **利润空间 (CPI-PPI)** | **鱼食/养分** | 企业赚不赚钱。卖价(CPI) - 成本(PPI) = 利润。养分足，鱼才长肉。 |
| **7** | **实体热度 (铜金比)** | **水温** | 经济是冷是热。铜代表工厂开工（热），金代表大家害怕存钱（冷）。 |
| **8** | **萨姆规则 (就业)** | **防洪警报** | **最终熔断器**。一旦失业率飙升，说明风暴已至，必须无条件撤退。 |

## 3. 实操指南：如何看懂看板信号？

看板通过**红绿灯**系统直接告诉你该做什么。你不需要懂复杂的数学公式，只需要看颜色和文字。

### 🚦 颜色信号系统
*   🟢 **绿色 (进攻)**：环境舒适，适合养鱼。**操作：买入核心资产、股票。**
*   🟡 **黄色 (观察)**：天气转阴，或者方向不明。**操作：多看少动，持有现金或收息资产。**
*   🔴 **红色 (防御)**：暴风雨、寒冬或干旱。**操作：卖出高风险资产，躲进避风港（黄金/美债/现金）。**

### 📊 八大维度详细解读

#### 1. 基准利率 EFFR (中央水龙头)
*   **对象**：
    *   **EFFR** : 美联储联邦基金有效利率 | **Fed Funds Rate**
        *   *比喻*：**中央水龙头**
        *   *数据*：`FRED:EFFR`
        *   *解释*：全社会资金的起始成本
    *   **EMA10** : 10个月指数移动平均线 | **10-Month EMA**
        *   *解释*：判断利率是处于上升还是下降趋势
*   **逻辑**：
    *   **降息周期 (EFFR 越低)**
        *   判断标准：EFFR < EMA10
        *   确认时长：FOMC 公布日
        *   含义：水龙头开大，钱变便宜，大家愿意借钱消费投资，鱼塘水位变高
    *   **加息周期 (EFFR 越高)**
        *   判断标准：EFFR > EMA10
        *   确认时长：FOMC 公布日
        *   含义：水龙头关紧，钱变贵，大家忙着还债不敢消费，鱼塘水位下降
*   **怎么做**：
    *   **EFFR < EMA10**：🟢 降息周期，水流变大。**进攻（买入科技股、成长股）**
    *   **EFFR > EMA10**：🔴 加息周期，水流枯竭。**防御（持有现金、短债）**

#### 2. 真实利率 TIPS (地心引力)
*   **对象**：
    *   **TIPS** : 10年期美国通胀保值债券收益率 | **10Y TIPS**
        *   *比喻*：**地心引力 / 躺赢收益**
        *   *数据*：`FRED:DFII10`
        *   *解释*：扣除通胀后的**真实**回报率。买它等于锁定“躺赢”的实际购买力
    *   **RGDP** : 美国**真实**GDP增长率 | **Real GDP**
        *   *比喻*：**奋斗收益**
        *   *数据*：`FRED:A191RL1Q225SBEA`
        *   *解释*：全社会辛辛苦苦干一年创造的财富增量（扣除通胀）。代表**奋斗者**的平均回报率
*   **逻辑**：
    *   **引力极大 (TIPS > RGDP)**
        *   判断标准：TIPS > RGDP
        *   确认时长：连续 5 日
        *   含义：躺赢比奋斗赚得多，全社会财富不够还债息。资金弃股买债
        *   结果：股市崩塌
    *   **引力适中 (0.5 RGDP < TIPS < RGDP)**
        *   判断标准：0.5 * RGDP < TIPS < RGDP
        *   确认时长：连续 5 日
        *   含义：躺赢收益尚可，资金观望
        *   结果：股市震荡
    *   **引力消失 (TIPS < 0.5 RGDP)**
        *   判断标准：TIPS < 0.5 * RGDP
        *   确认时长：连续 5 日
        *   含义：躺赢不如奋斗，资金被迫去投资股票房产博取收益
        *   结果：股市起飞
*   **怎么做**：
    *   **TIPS < 0.5 * RGDP**：🟢 引力小。**买入（科技股、核心资产）**
    *   **TIPS > RGDP**：🔴 崩坏信号！**清仓（卖出所有股票，换成现金或黄金）**
    *   **中间态**：🟡 震荡期。**观望（持有高股息资产，不折腾）**

#### 3. 利差结构 10Y-2Y (输水管道)
*   **对象**：
    *   **10Y** : 10年期美国国债收益率 | **10年美债**
        *   *比喻*：**未来信心**
        *   *数据*：`FRED:GS10`
        *   *解释*：代表市场对美国长期经济增长和通胀的预期
    *   **2Y** : 2年期美国国债收益率 | **2年美债**
        *   *比喻*：**当下政策**
        *   *数据*：`FRED:GS2`
        *   *解释*：代表市场对美联储近期加息/降息动作的预期
*   **逻辑**：
    *   **正常 (10Y > 2Y)**
        存钱越久利息越高，水往低处流，管道通畅
    *   **倒挂 (10Y < 2Y)**
        存久了反而利息低，说明大家觉得未来经济很差（比现在还差），水倒流了
    *   **牛陡 (Bull Steep)**
        *   判断标准：10Y 和 2Y 均下跌，但 2Y 跌幅 > 10Y 跌幅 (导致利差变大，曲线变陡)
        *   确认时长：即时生效 (基于 20 日趋势)
        *   含义：最危险的信号。意思是大家在赌央行必须马上救命（降息），所以短债利率暴跌。这说明管道爆裂了，危机已经发生
    *   **熊陡 (Bear Steep)**
        *   判断标准：10Y 和 2Y 均上涨，但 10Y 涨幅 > 2Y 涨幅 (导致利差变大，曲线变陡)
        *   确认时长：即时生效 (基于 20 日趋势)
        *   含义：意思是长债利率涨得快。这说明大家对未来经济复苏有信心，要求更高的长期回报
    *   **平坦 (Flat)**
        利差接近0，或者变化不明显。说明市场方向未明
*   **怎么做**：
    *   **牛陡**：🔴 看到这个词，**立刻清仓（全部卖出）**
    *   **倒挂**：🔴 衰退预警。**防御（持有现金）**
    *   **熊陡**：🟢 经济复苏。**买入（大宗商品、顺周期股）**
    *   **平坦**：🟡 观望。**不操作**

#### 4. 信用风险 OAS (河道淤泥)
*   **对象**：
    *   **OAS** : 美国高收益企业债期权调整利差 | **High Yield Spread**
        *   *比喻*：**河道淤泥 / 银行的恐惧值**
        *   *数据*：`FRED:BAMLH0A0HYM2`
        *   *解释*：银行借钱给信用不好的公司时多收的利息。利差越大，说明银行越害怕公司倒闭
    *   **斜率 | Slope** : 20日线性回归斜率
        *   *解释*：判断信用环境是在恶化还是改善
*   **逻辑**：
    *   **环境恶化 (斜率向上)**
        *   判断标准：斜率 > 0
        *   确认时长：连续 3 日
        *   含义：银行越来越害怕，不敢借钱，河道淤泥堆积
    *   **环境改善 (斜率向下)**
        *   判断标准：斜率 < 0
        *   确认时长：连续 3 日
        *   含义：银行胆子变大，愿意放贷，河道疏通
    *   **极度危险 (OAS > 10%)**
        *   判断标准：OAS > 10%
        *   确认时长：即时生效
        *   含义：金融危机爆发
*   **怎么做**：
    *   **OAS < 3.5%**：🟢 淤泥少，借钱容易。**进攻（买入股票）**
    *   **OAS > 6.0%**：🔴 淤泥堵塞，会有公司破产。**防御（不碰高杠杆公司，持有现金）**
    *   **OAS > 10.0%**：🔴 **金融危机确认**。快跑！
    *   **斜率 < 0**：🟢 超跌修复。**买入（博弈反弹）**

#### 5. 流动性 (蓄水池水位)

*   **为什么中美逻辑不同？**
    *   🇺🇸 **美国 (全球水龙头)**：
        *   **直接融资**：美联储印钱 -> 华尔街拿到钱 -> 投入股市/债市。
        *   **全球收割**：美元是世界货币，全球资金（日本/欧洲/中东）都会买美债美股。
        *   **M2 的意义**：只要 M2 涨，意味着全球流动性泛滥，水一定会流进美股，推高估值。
    *   🇨🇳 **中国 (内部堰塞湖)**：
        *   **间接融资**：央行印钱 -> 商业银行拿到钱 -> 贷款给企业。
        *   **资本管制**：钱主要在国内转，流不出去，也难进来。
        *   **剪刀差的意义**：钱到了银行，如果贷不出去（企业不愿借），就形成堰塞湖。只有 **M1 增速 > M2 增速**，才说明堰塞湖开了口子，活钱流向了实体和股市。
        *   **房地产**：以前房子是印钞机（抵押贷款创造 M2），现在房子熄火，印钞机停了，所以 M1 负增长。

##### 🇺🇸 美国 (M2 真实供应)
*   **对象**：
    *   **M2** : 广义货币供应量（剔除通胀）| **Real M2**
        *   *比喻*：**蓄水池水位**
        *   *数据*：`FRED:M2REAL`
        *   *解释*：市面上所有钱的总和。钱是资产价格的燃料
    *   **M2_YoY** : M2 同比增速 | **M2 Growth**
        *   *解释*：判断资金是在变多还是变少
*   **逻辑**：
    > **确认时长**：连续 2 个月
    *   **水位上涨 (正增长)**
        *   判断标准：M2_YoY > 0.2%
        *   含义：实际货币供应扩张，资金充裕
    *   **水位下降 (负增长)**
        *   判断标准：M2_YoY < -0.2%
        *   含义：实际货币供应收缩，资金枯竭
    *   **存量博弈 (持平)**
        *   判断标准：介于上述两者之间
        *   含义：水位变化不大
*   **怎么做**：
    *   **水位上涨**：🟢 资金充裕。**买入（科技股、成长股）**
    *   **水位下降**：🔴 资金枯竭。**空仓（持有现金）**
    *   **中间态**：🟡 震荡。**观望**

##### 🇨🇳 中国 (M1-M2 剪刀差)
*   **对象**：
    *   **M1** : 狭义货币供应量
        *   *数据*：`CN:M1`
        *   *比喻*：**企业活钱**
        *   *解释*：企业的活钱，代表投资意愿。M1 高说明企业准备大干一场
    *   **M2** : 广义货币供应量
        *   *数据*：`CN:M2`
        *   *比喻*：**社会总钱**
        *   *解释*：全社会的钱（含定期存款）。M2 高但 M1 低，说明钱都在睡觉
    *   **剪刀差** : M1增速 - M2增速
        *   *比喻*：**信贷脉搏**
        *   *解释*：判断钱是流向实体经济（正剪刀差）还是在银行空转（负剪刀差）
*   **逻辑**：
    > **确认时长**：连续 2 个月
    *   **信贷脉搏激活 (正剪刀差)**
        *   判断标准：M1增速 > M2增速
        *   含义：资金活化，企业愿意投资，利好股市
    *   **流动性陷阱 (负剪刀差)**
        *   判断标准：M1增速 < M2增速
        *   含义：资金空转，企业躺平存钱，利空股市
*   **怎么做**：
    *   **脉搏激活**：🟢 资金活化。**买入（科技股、成长股）**
    *   **陷阱确认**：🔴 资金空转。**空仓（持有现金）**

#### 6. 利润空间 (鱼食养分)

*   **为什么中美逻辑不同？**
    *   🇺🇸 **美国 (全球最大消费国)**：
        *   **逻辑闭环**：美元霸权 -> 全球廉价商品输入 -> 消费驱动经济 (GDP占比 70%)。
        *   **CPI 的意义**：因为大部分商品靠进口，成本相对固定（输入性）。如果 CPI 涨，说明**老百姓手里有钱，买得起**，需求非常旺盛。企业看到大家抢着买，就敢涨价，利润自然就厚了。
    *   🇨🇳 **中国 (全球最大制造国)**：
        *   **逻辑闭环**：全产业链制造 -> 产能巨大 -> 出口驱动经济。
        *   **PPI 的意义**：因为是世界工厂，产能往往过剩。PPI 代表**工厂出厂价**。如果 PPI 跌，说明东西卖不出去，只能降价互砍（内卷）。只有 PPI 涨，才说明全球都在买中国货，工厂日子才好过。

##### 🇺🇸 美国 (CPI-PPI)
*   **对象**：
    *   **CPI** : 消费者价格指数
        *   *比喻*：**卖价**
        *   *数据*：`FRED:CPIAUCSL`
        *   *解释*：代表终端消费品的售价。如果 CPI 高，说明企业能把产品卖个好价钱
    *   **PPI** : 生产者价格指数
        *   *比喻*：**成本**
        *   *数据*：`FRED:PPIACO`
        *   *解释*：代表原材料和出厂成本。如果 PPI 高，说明企业进货贵，利润被吃掉
    *   **剪刀差** : CPI - PPI
        *   *解释*：企业的真实利润空间。卖价涨得比成本快，企业才赚钱
*   **逻辑**：
    > **确认时长**：连续 2 个月 (内卷需 3 个月)
    *   **利润释放 (走阔)**
        *   判断标准：剪刀差 > 前值
        *   含义：需求旺盛，企业提价能力强
    *   **内卷严重 (收窄)**
        *   判断标准：剪刀差 < 前值
        *   含义：通胀回落，定价权丧失
    *   **亏本赚吆喝 (倒挂)**
        *   判断标准：PPI > CPI
        *   含义：成本高于售价，企业亏损
*   **怎么做**：
    *   **剪刀差走阔**：🟢 利润释放。**买入（行业龙头股）**
    *   **剪刀差收窄**：🟡 通胀回落。**观望（等待通胀见底）**
    *   **剪刀差倒挂**：🔴 成本倒挂。**防御（回避制造业）**

##### 🇨🇳 中国 (CPI-PPI)
*   **对象**：
    *   **CPI** : 消费者价格指数
        *   *比喻*：**下游需求**
        *   *数据*：`CN:CPI`
        *   *解释*：反映老百姓买不买账。如果 CPI 持续低迷，说明大家不愿意花钱
    *   **PPI** : 生产者价格指数
        *   *比喻*：**上游产能**
        *   *数据*：`CN:PPI`
        *   *解释*：反映工厂出厂价。如果 PPI 持续为负，说明产能过剩，工厂在亏本甩卖
    *   **剪刀差** : CPI - PPI
        *   *解释*：产业链利润分配。剪刀差走阔，说明中下游日子好过；收窄说明上游挤压下游
*   **逻辑**：
    > **确认时长**：连续 2 个月 (内卷需 3 个月)
    *   **利润释放 (走阔)**
        *   判断标准：剪刀差 > 前值
        *   含义：上游成本下降，中下游利润改善
    *   **内卷严重 (收窄)**
        *   判断标准：剪刀差 < 前值
        *   含义：产能过剩，价格战激烈
    *   **通缩风险 (双负)**
        *   判断标准：CPI < 0 且 PPI < 0
        *   含义：全产业链通缩，需求疲软
*   **怎么做**：
    *   **剪刀差走阔**：🟢 利润改善。**买入（中下游制造股）**
    *   **剪刀差收窄**：🟡 价格内卷。**观望（回避同质化竞争）**
    *   **通缩风险**：🔴 全线通缩。**防御（持有国债/现金）**

#### 7. 实体热度 铜金比 (水温)
*   **对象**：
    *   **铜 | Copper** : 铜期货价格
        *   *比喻*：**开工需求**
        *   *数据*：`COMEX:HG1!`
    *   **金 | Gold** : 黄金期货价格

### 🧠 3. 宏观底层逻辑 (进阶心法)

这里是决定资产价格的“隐形手”，读懂这些，你才能真正理解上面的八大维度。

#### 1. 美元潮汐 (收割全球)
*   **霸权根基**：
    *   **锚定刚需**：以前锚定黄金，现在锚定**石油**。你想买石油，就必须用美元。
    *   **无本万利**：美国印绿纸买全球商品（贸易逆差）-> 全球赚了美元没处花 -> 买回美债（资金回流）。**本质：美国白嫖了商品，还让你成为债主。**
*   **收割原理**：
    *   **放水 (降息)**：美元便宜，流向新兴市场炒高资产。
    *   **收水 (加息)**：美元变贵回流，新兴市场缺钱崩盘，美国再来抄底。
*   **对中国的影响**：美联储加息时，中国为了留住资金（不让外资跑光），很难大幅降息。这就是为什么美联储加息，A股也难受。

#### 2. 不可能三角 (政策两难)
*   **原理**：一个国家不可能同时实现以下三点：
    1.  **资本自由流动**（钱想走就走）
    2.  **汇率稳定**（人民币不贬值）
    3.  **货币政策独立**（想降息就降息）
*   **为什么不可能？**：假设你要 **1** 和 **3**（自由流动 + 独立降息）。当你降息时，钱为了逐利会大量外逃（因为自由流动），导致汇率崩盘。所以，如果你想保住 **2**（汇率），就必须在 **1** 和 **3** 之间二选一。
*   **为什么要保汇率 (2)？**
    *   **防通胀**：中国大量进口石油粮食。汇率崩盘 = 进口成本翻倍 = 物价飞涨 = 老百姓吃不起饭。
    *   **防破产**：房企借了很多美元债。汇率崩盘 = 还债成本翻倍 = 企业破产 = 金融危机。
*   **中国的选择**：为了保 **2** 和 **3**，只能牺牲 **1**（每人每年 5 万美元额度，就是为了防止富人把资产搬空）。
*   **实战：中美利差 (10Y-10Y)**
    *   **定义**：10年美债收益率 - 10年中国国债收益率。
    *   **逻辑**：资本逐利。如果美债利息远高于中国（**倒挂**），资金就会卖人民币买美元。
    *   **影响**：为了保汇率，中国央行不敢大幅降息。**结果：A股流动性起不来。**

#### 3. 债务周期 (达里奥)
*   **原理**：经济的好坏，长期看**生产率**，短期看**债务**。
    *   **加杠杆 (借钱)**：借钱消费/投资 -> 需求旺盛 -> 资产涨。**（繁荣期）**
    *   **去杠杆 (还钱)**：还债 -> 消费/投资减少 -> 需求萎缩 -> 资产跌。**（衰退期）**
*   **现状**：
    *   **美国**：处于再通胀周期（居民资产负债表健康，还在借钱买买买）。
    *   **中国**：处于去杠杆周期（房地产/城投债压力大，大家都在还钱，不愿意借钱）。
*   **结论**：这就是为什么美国看通胀（防过热），中国防通缩（防过冷）。

    *   **含义**：震荡市，或者变盘点。
    *   **操作**：**猥琐发育**。不要上杠杆，可以定投，但要留足子弹。
