## 1. 写在前面：为什么你需要这个看板？

投资世界里充满了噪音。新闻说“经济很好”，股市却大跌；专家说“要降息”，手里的基金却还在亏。

这个看板的设计初衷，就是**剥离所有噪音，只看最核心的“水流”和“天气”**。我们将复杂的宏观经济简化为一个**“生态鱼塘”模型**，帮助你判断：
- **现在适合养鱼吗？**（进攻/买入）
- **还是暴风雨要来了？**（防守/卖出）

## 2. 核心概念：宏观经济的“生态鱼塘”比喻

想象你是一个鱼塘主（投资者），你的钱就是鱼。鱼能不能养好，取决于以下 8 个维度的环境指标：

| 维度 | 指标名称 | 鱼塘比喻 | 解释 |
| :--- | :--- | :--- | :--- |
| **1** | **基准利率 (EFFR)** | **中央水龙头** | 央行控制水流的总开关。利率越低，水流越大（钱越便宜），鱼塘水位越高。 |
| **2** | **真实利率 (TIPS)** | **地心引力** | 决定资产起飞的难度。引力越大（利率高），资产价格越难涨；引力小，猪都能上天。 |
| **3** | **利差结构 (10Y-2Y)** | **输水管道** | 管道是否通畅。正常水往低处流（长债利率>短债）。如果倒流（倒挂）或爆管（牛陡），说明系统出故障了。 |
| **4** | **信用风险 (OAS)** | **河道淤泥** | 银行愿不愿意借钱给企业。淤泥多（利差大），水流不过去，企业会渴死。 |
| **5** | **流动性 (M2)** | **蓄水池水位** | 整个市场的钱是多了还是少了。水位上涨，鱼才欢腾。 |
| **6** | **利润空间 (CPI-PPI)** | **鱼食/养分** | 企业赚不赚钱。卖价(CPI) - 成本(PPI) = 利润。养分足，鱼才长肉。 |
| **7** | **实体热度 (铜金比)** | **水温** | 经济是冷是热。铜代表工厂开工（热），金代表大家害怕存钱（冷）。 |
| **8** | **萨姆规则 (就业)** | **防洪警报** | **最终熔断器**。一旦失业率飙升，说明风暴已至，必须无条件撤退。 |

## 3. 实操指南：如何看懂看板信号？

看板通过**红绿灯**系统直接告诉你该做什么。你不需要懂复杂的数学公式，只需要看颜色和文字。

### 🚦 颜色信号系统
*   🟢 **绿色 (进攻)**：环境舒适，适合养鱼。**操作：买入核心资产、股票。**
*   🟡 **黄色 (观察)**：天气转阴，或者方向不明。**操作：多看少动，持有现金或收息资产。**
*   🔴 **红色 (防御)**：暴风雨、寒冬或干旱。**操作：卖出高风险资产，躲进避风港（黄金/美债/现金）。**

### 📊 八大维度详细解读

#### 1. 基准利率 EFFR (中央水龙头)
*   **对象**：
    *   **基准利率 | EFFR** : 美联储联邦基金有效利率。
        *   *比喻*：**中央水龙头**。
        *   *数据*：`FRED:EFFR`
        *   *解释*：央行控制着全社会资金的总闸门。它决定了银行之间借钱的利息，也决定了你房贷车贷的基准利息。它是**全社会资金的起始成本**。
*   **逻辑**：
    *   **降息周期 (EFFR 越低)**
        **判断标准：EFFR数值 < 10个月移动平均线**。
        **确认时长：FOMC公布日**。
        **含义**：水龙头开大，钱变便宜，大家愿意借钱消费投资，鱼塘水位变高。
    *   **加息周期 (EFFR 越高)**
        **判断标准：EFFR数值 > 10个月移动平均线**。
        **确认时长：FOMC公布日**。
        **含义**：水龙头关紧，钱变贵，大家忙着还债不敢消费，鱼塘水位下降。
*   **怎么做**：
    *   **EFFR < 10月均线**：🟢 降息周期，水流变大。**进攻（买入科技股、成长股）**。
    *   **EFFR > 10月均线**：🔴 加息周期，水流枯竭。**防御（持有现金、短债）**。

#### 2. 真实利率 TIPS (地心引力)
*   **对象**：
    *   **真实利率 | TIPS** : 10年期通胀保值债券。
        *   *比喻*：**地心引力 / 躺赢收益**。
        *   *数据*：`FRED:DFII10`
        *   *解释*：这是美国政府发行的债券。因为它“保值”（收益率已经扣除了通胀），所以你买它就等于**躺着不动也能稳赚的真实购买力**。
    *   **美国真实GDP | RGDP** : 美国实际国内生产总值。
        *   *比喻*：**奋斗收益**。
        *   *数据*：`FRED:A191RL1Q225SBEA`
        *   *解释*：全社会大家辛辛苦苦干一年创造的平均财富增量（已扣除通胀）。它代表了**奋斗者的平均回报率**。
*   **逻辑**：
    *   **引力极大 (TIPS > RGDP)**
        **判断标准：TIPS收益率 > RGDP增长率**。
        **确认时长：连续5日**。
        **含义**：躺赢比奋斗赚得多，全社会财富不够还债息。资金弃股买债。
        **结果：股市崩塌**。
    *   **引力适中 (0.5 RGDP < TIPS < RGDP)**
        **判断标准：TIPS介于RGDP的一半和全值之间**。
        **确认时长：连续5日**。
        **含义**：躺赢收益尚可，资金观望。
        **结果：股市震荡**。
    *   **引力消失 (TIPS < 0.5 RGDP)**
        **判断标准：TIPS收益率 < 0.5倍RGDP增长率**。
        **确认时长：连续5日**。
        **含义**：躺赢不如奋斗，资金被迫去投资股票房产博取收益。
        **结果：股市起飞**。
*   **怎么做**：
    *   **TIPS < 0.5 RGDP**：🟢 引力小。**买入（科技股、核心资产）**。
    *   **TIPS > RGDP**：🔴 崩坏信号！**清仓（卖出所有股票，换成现金或黄金）**。
    *   **中间态**：🟡 震荡期。**观望（持有高股息资产，不折腾）**。

#### 3. 利差结构 10Y-2Y (输水管道)
*   **对象**：
    *   **10年期美债 | 10Y** : 10年期国债收益率。
        *   *比喻*：**未来信心**。
        *   *数据*：`FRED:GS10`
        *   *解释*：大家觉得未来经济好，才会买10年后的债。
    *   **2年期美债 | 2Y** : 2年期国债收益率。
        *   *比喻*：**当下政策**。
        *   *数据*：`FRED:GS2`
        *   *解释*：大家觉得央行马上要加息还是降息。
*   **逻辑**：
    *   **正常 (10Y > 2Y)**
        存钱越久利息越高，水往低处流，管道通畅。
    *   **倒挂 (10Y < 2Y)**
        存久了反而利息低，说明大家觉得未来经济很差（比现在还差），水倒流了。
    *   **牛陡 (Bull Steep)**
        **判断标准：10Y和2Y均下跌，但2Y跌幅 > 10Y跌幅 (导致利差变大，曲线变陡)**。
        **确认时长：即时生效 (基于20日趋势)**。
        **含义**：最危险的信号。意思是大家在赌央行必须马上救命（降息），所以短债利率暴跌。这说明管道爆裂了，危机已经发生。
    *   **熊陡 (Bear Steep)**
        **判断标准：10Y和2Y均上涨，但10Y涨幅 > 2Y涨幅 (导致利差变大，曲线变陡)**。
        **确认时长：即时生效 (基于20日趋势)**。
        **含义**：意思是长债利率涨得快。这说明大家对未来经济复苏有信心，要求更高的长期回报。
    *   **平坦 (Flat)**
        利差接近0，或者变化不明显。说明市场方向未明。
*   **怎么做**：
    *   **牛陡**：🔴 看到这个词，**立刻清仓（全部卖出）**。
    *   **倒挂**：🔴 衰退预警。**防御（持有现金）**。
    *   **熊陡**：🟢 经济复苏。**买入（大宗商品、顺周期股）**。
    *   **平坦**：🟡 观望。**不操作**。

#### 4. 信用风险 OAS (河道淤泥)
*   **对象**：
    *   **高收益债利差 | OAS** : 美元高收益企业债期权调整利差。
        *   *比喻*：**河道淤泥 / 银行的恐惧值**。
        *   *数据*：`FRED:BAMLH0A0HYM2`
        *   *解释*：这是银行借钱给信用不好的公司时，额外多收的利息。利息收得越高，说明银行越害怕这家公司倒闭。
*   **逻辑**：
    *   **环境恶化 (斜率向上)**
        **判断标准：20日线性回归斜率 > 0**。
        **确认时长：连续3日**。
        **含义**：银行越来越害怕，不敢借钱，河道淤泥堆积。
    *   **环境改善 (斜率向下)**
        **判断标准：20日线性回归斜率 < 0**。
        **确认时长：连续3日**。
        **含义**：银行胆子变大，愿意放贷，河道疏通。
    *   **极度危险 (OAS > 10%)**
        **判断标准：OAS数值 > 10%**。
        **确认时长：即时生效**。
        **含义**：金融危机爆发。
*   **怎么做**：
    *   **OAS < 3.5%**：🟢 淤泥少，借钱容易。**进攻（买入股票）**。
    *   **OAS > 6.0%**：🔴 淤泥堵塞，会有公司破产。**防御（不碰高杠杆公司，持有现金）**。
    *   **OAS > 10.0%**：🔴 **金融危机确认**。快跑！
    *   **斜率变好**：🟢 超跌修复。**买入（博弈反弹）**。

#### 5. 流动性 M2 (蓄水池水位)
*   **对象**：
    *   **真实M2供应量 | M2** : 广义货币供应量（剔除通胀）。
        *   *比喻*：**蓄水池水位**。
        *   *数据*：`FRED:M2REAL`
        *   *解释*：市面上所有钱的总和（扣除通胀后）。钱是资产价格的燃料。
*   **逻辑**：
    *   **水位上涨 (正增长)**
        **判断标准：M2同比增速 > 0.2%**。
        **确认时长：连续2个月**。
        **含义**：钱变多了，水涨船高，利好股市。
    *   **水位下降 (负增长)**
        **判断标准：M2同比增速 < -0.2%**。
        **确认时长：连续2个月**。
        **含义**：钱变少了，船会搁浅，利空股市。
    *   **存量博弈 (持平)**
        **判断标准：M2同比增速在 -0.2% 到 0.2% 之间**。
        **确认时长：连续2个月**。
        **含义**：水位变化不大。
*   **怎么做**：
    *   **M2 > 0.2**：🟢 水位上涨。**买入（科技股、成长股）**。
    *   **M2 < -0.2**：🔴 水位下降。**空仓（持有现金）**。
    *   **中间态**：🟡 震荡。**观望**。

#### 6. 利润空间 CPI-PPI (鱼食养分)
*   **对象**：
    *   **消费者物价 | CPI** : 城市消费者价格指数。
        *   *比喻*：**卖价**。
        *   *数据*：`FRED:CPIAUCSL`
    *   **生产者物价 | PPI** : 生产者价格指数。
        *   *比喻*：**成本**。
        *   *数据*：`FRED:PPIACO`
*   **逻辑**：
    *   **利润释放 (走阔)**
        **判断标准：(CPI - PPI) 差值 > 前值**。
        **确认时长：连续2个月**。
        **含义**：卖得贵成本低，企业赚钱容易，养分足。
    *   **内卷严重 (收窄)**
        **判断标准：连续3个月差值 < 前值**。
        **确认时长：连续3个月**。
        **含义**：卖不上价成本高，企业赚钱难，养分少。
    *   **亏本赚吆喝 (倒挂)**
        **判断标准：PPI > CPI**。
        **确认时长：连续2个月**。
        **含义**：成本比售价还高，企业亏损。
*   **怎么做**：
    *   **走阔**：🟢 利润释放。**买入（行业龙头股）**。
    *   **收窄/连续变差**：🟡 内卷严重。**观望（少动）**。
    *   **倒挂 (PPI > CPI)**：🔴 亏本赚吆喝。**防御**。

#### 7. 实体热度 铜金比 (水温)
*   **对象**：
    *   **铜 | Copper** : 铜期货价格。
        *   *比喻*：**开工需求**（经济好才买铜）。
        *   *数据*：`COMEX:HG1!`
    *   **金 | Gold** : 黄金期货价格。
        *   *比喻*：**恐惧情绪**（害怕才买金）。
        *   *数据*：`COMEX:GC1!`
*   **逻辑**：
    *   **水温适宜 (铜强金弱)**
        **判断标准：铜价涨幅 > 金价涨幅**。
        **确认时长：连续5日**。
        **含义**：大家忙着开工，没人害怕。
    *   **水温冰冷 (金强铜弱)**
        **判断标准：金价涨幅 > 铜价涨幅**。
        **确认时长：连续5日**。
        **含义**：大家不敢开工，都在买金避险。
    *   **危机前兆 (暴跌)**
        **判断标准：单日跌幅 > 3%**。
        **确认时长：即时生效**。
        **含义**：需求突然冻结。
*   **怎么做**：
    *   **铜强金弱**：🟢 经济火热。**买入（股票、商品）**。
    *   **金强铜弱**：🔴 实体寒冬。**买入（黄金、国债）**。
    *   **背离**：🟡 方向未明。**观望**。

#### 8. 萨姆规则 (防洪警报)
*   **对象**：
    *   **萨姆规则 | Sahm Rule** : 实时萨姆衰退指标。
        *   *比喻*：**最终熔断器 / 死神信号**。
        *   *数据*：`FRED:SAHMREALTIME`
        *   *解释*：一旦失业率开始快速上升（比过去12个月低点高出0.5%），说明经济已经刹不住车了，必然进入衰退。这是历史上准确率 100% 的死神信号。
*   **逻辑**：
    *   **确认衰退 (> 0.5%)**
        **判断标准：(失业率3月均值 - 12月低点) > 0.5%**。
        **确认时长：即时生效**。
        **含义**：经济刹不住车，必然衰退。
    *   **危险边缘 (0.3% - 0.5%)**
        **判断标准：数值在 0.3% 到 0.5% 之间**。
        **确认时长：当月确认**。
        **含义**：就业市场正在恶化。
    *   **就业强劲 (< 0.3%)**
        **判断标准：数值 < 0.3%**。
        **确认时长：当月确认**。
        **含义**：就业市场健康。
*   **怎么做**：
    *   **触发熔断 (>0.5)**：🔴 只要看到这个信号，别管其他指标多好，**立刻清仓（卖出所有风险资产）**。保命要紧。
    *   **危险边缘**：🟡 谨慎。**减仓**。

## 4. 懒人操作法：如何做决策？

别被这么多指标吓到，你只需要看看板左下角的 **【最终诊断】**：

1.  **看到【熔断】字样（深红色背景）**：
    *   **含义**：萨姆规则触发、信用崩塌或牛陡救火。
    *   **操作**：**一键清仓**。持有现金或黄金。别抱侥幸心理。

2.  **总分 > 6.0 (全绿)**：
    *   **含义**：顺风顺水，全方位复苏。
    *   **操作**：**满仓进攻**。买科技巨头、核心资产。

3.  **总分 < -2.0 (红多绿少)**：
    *   **含义**：环境恶劣。
    *   **操作**：**防守**。50%现金 + 50%高股息/红利股。

4.  **总分在 -2.0 到 6.0 之间 (黄/混色)**：
    *   **含义**：震荡市，或者变盘点。
    *   **操作**：**猥琐发育**。不要上杠杆，可以定投，但要留足子弹。
